Enum KryptonPageFlags

Specifies flags that can be applied to a KryptonPage

Namespace: System.Dynamic.ExpandoObject
Assembly: ComponentFactory.Krypton.Navigator.dll
Syntax
[Flags]
public enum KryptonPageFlags

All

Specifies that all flags are set.

AllowConfigSave

Specifies that the page is allowed to be saved to configuration.

AllowPageDrag

Specifies that the page is allowed to be dragged from the navigator.

AllowPageReorder

Specifies that the page is allowed to be drag reordered.

DockingAllowAutoHidden

Specifies that the user is allowed to make a page auto hidden.

DockingAllowClose

Specifies that the user is allowed to close a docking page.

DockingAllowDocked

Specifies that the user is allowed to make a page docked.

DockingAllowDropDown

Specifies that the user is allowed to select from a drop down button.

DockingAllowFloating

Specifies that the user is allowed to make a page floating.

DockingAllowNavigator

Specifies that the user is allowed to make a page tabbed in a navigator.

DockingAllowWorkspace

Specifies that the user is allowed to make a page tabbed in a workspace.

PageInOverflowBarForOutlookMode

Specifies that in the Outlook mode the page is shown on the overflow bar.
