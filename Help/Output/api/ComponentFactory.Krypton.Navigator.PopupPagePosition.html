Enum PopupPagePosition

Specifies how to auto calculate the popup page position.

Namespace: System.Dynamic.ExpandoObject
Assembly: ComponentFactory.Krypton.Navigator.dll
Syntax
[TypeConverter(typeof (PopupPagePositionConverter))]
public enum PopupPagePosition

AboveFar

Specifies the popup is above the relative item and aligned to far edge.

AboveMatch

Specifies the popup is above the relative item and same width as the relative item.

AboveNear

Specifies the popup is above the relative item and aligned to near edge.

Specifies the popup is below the relative item and aligned to far edge.

BelowMatch

Specifies the popup is below the relative item and same width as the relative item.
